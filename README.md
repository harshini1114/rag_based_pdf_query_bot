## Flask + LangChain + ChromaDB RAG Chatbot

A minimal, production-ready Retrieval-Augmented Generation (RAG) chatbot built with Flask, LangChain, ChromaDB, and OpenAI. 
The system retrieves relevant documents from a vector database and uses an LLM to generate grounded answers.

## Starting Application....

- python3 -m venv venv
- source ./venv/bin/activate
- pip3 install -r requirements.txt
- python3 flask_app.py
- Server runs at http://127.0.0.1:5000
